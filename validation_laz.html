About

The riskmetric package is a collection of risk metrics to evaluate the quality of R packages. The risk of using an R package is evaluated based on a number of metrics meant to evaluate development best practices, code documentation, community engagement and development sustainability.

The oysterR package scans R packages with the OSS Index provided by Sonatype and reports any known vulnerabilities.
